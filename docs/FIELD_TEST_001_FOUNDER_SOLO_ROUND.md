# FIELD TEST-001 — Founder Solo Round

**Status: Planning Document (No Implementation)**

이 문서는 실제 라운드 중 Founder가 직접 채우는 관찰 기록 템플릿이다. Claude는 이 관찰 항목을 대신 채우지 않는다 — 실제 손목의 Watch, 실제 필드에서의 행동은 시뮬레이션 대상이 아니다.

---

## 목적

이번 테스트의 목적은 버그를 찾는 것이 아니다.

Founder가 실제 라운드에서 **언제 / 왜 / 얼마나** FIELDTALK를 사용하는지를 관찰하는 것이다.

기능 평가보다 **사용 행동을 기록하는 것**이 우선이다.

---

## Test Rule

테스트 중에는 억지로 앱을 사용하지 않는다. 사용하고 싶을 때만 사용한다.

**사용하지 않았다면, 그것도 중요한 결과다.**

---
